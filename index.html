About The Project

This project demonstrates a simple face recognition application using cutting edge AI technologies. It is designed to identify and verify individuals using biometric facial features with high accuracy and speed. Technologies used: HTML, CSS, JavaScript, face-api.js Framework: React

How It Works

Step 1

Upload or provide a face image.

Step 2

Our AI analyses the image and detect facial features.

Step 3

Results are displayed in a simple , easy to understand format.
